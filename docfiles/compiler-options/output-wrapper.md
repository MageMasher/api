## Summary

Wrap the JavaScript output in an [IIFE] `(function(){…​};)()` to avoid
clobbering globals. Defaults to false.

[IIFE]:https://en.wikipedia.org/wiki/Immediately-invoked_function_expression

```clj
:output-wrapper true
```
