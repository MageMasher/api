## Summary

## Details

Replaces all instance of `match` with `replacement` in `s`.

The options for match / replacement are:

| match  | replacement |
|--------|-------------|
| string | string      |
| regex  | string      |
| regex  | function    |

## Examples

## Signature
[s match replacement]
