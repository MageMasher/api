## cljs.js

 <table border="1">
<tr>
<td>namespace</td>
<td><a href="https://github.com/cljsinfo/cljs-api-docs/tree/1.7.10"><img valign="middle" alt="[+] 1.7.10" title="Added in 1.7.10" src="https://img.shields.io/badge/+-1.7.10-lightgrey.svg"></a> </td>
</tr>
</table>

This is what is referred to as the &quot;Bootstrapped Compiler&quot;, which allows
you to compile ClojureScript with nothing but ClojureScript.  In other
words, this namespace provides the ability to evaluate ClojureScript
code at runtime.
